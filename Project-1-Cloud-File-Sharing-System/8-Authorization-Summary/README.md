# Authorization Summary
The Cloud File Sharing System (CFSS) has a moderate level of risk, which is consistent with its system categorization. The assessment found a number of control gaps, mostly having to do with managing access, monitoring practices, and relying on third-party services. These gaps don't mean that security controls don't work at all; they just show where they could be better.

Based on the results of this assessment, it is recommended that the system be considered for conditional authorization, pending the fixing of the weaknesses that were found in the Plan of Action and Milestones (POA&M). The designated Authorizing Official is still in charge of making final decisions about authorization.
