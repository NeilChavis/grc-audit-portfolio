# FIPS 199 Categorization
The Cloud File Sharing System (CFSS) was put into groups according to FIPS 199 to find out how a security breach could affect the confidentiality, integrity, and availability. This classification shows what kind of data the system handles and how it helps Apex Consulting Group run its company.

## Security Categorization
| Security Objective | Impact Level |
| ------------------ | ------------ |
| Confidentiality    | Moderate     |
| Integrity          | Moderate     |
| Availability       | Moderate     |

## Justification
### Confidentiality: Moderate
The unauthorized disclosure of information within the Cloud File Sharing System (CFSS) would result in a serious adverse effect on the organization. The system contains business-sensitive data, financial information, and personally identifiable information (PII), which could lead to loss of client trust, reputational damage, and potential privacy concerns if exposed. However, the impact is not expected to be severe or catastrophic to organizational operations.
### Integrity: Moderate
The unauthorized modification or corruption of data within the Cloud File Sharing System (CFSS) would result in a serious adverse effect on the organization. Inaccurate or altered client deliverables, financial records, or internal documents could negatively impact business operations, decision-making, and organizational credibility. However, such impacts are not expected to result in severe or catastrophic failure of core organizational functions.
### Availability: Moderate
The loss of availability of the Cloud File Sharing System (CFSS) would result in a serious adverse effect on the organization. Users would be unable to access critical files and collaborate effectively, leading to delays in project execution and reduced operational efficiency. However, the organization would still be able to continue operations through alternative means, and the impact would not be severe or catastrophic.
