# Risk Register
The risk register below lists the most important concerns that were found during the evaluation of the Cloud File Sharing System (CFSS). We looked at risks based on how likely they were to happen and how detrimental they may be for the system's operating environment.

Risk ratings are based on qualitative assessment using High, Moderate, and Low impact and likelihood levels.
