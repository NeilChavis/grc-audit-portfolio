# Project 1: Cloud File Sharing System
## Overview
This project demonstrates a GRC / ISSO style system security authorization and risk assessment effort for a Cloud File Sharing System operating in a SaaS environment.

The project is designed to simulate realistic RMF-based security documentation and analysis activities that align with work commonly performed by GRC Analysts and ISSOs.

## Project Scope
The project focuses on:
- system overview and boundary definition
- shared responsibility in a SaaS environment
- FIPS 199 categorization
- control selection using NIST SP 800-53
- control implementation statements
- risk identification and analysis
- POA&M development
- authorization recommendation summary

## Repository Structure
- 1-System-Overview
- 2-Cloud-Responsibility
- 3-FIPS-199-Categorization
- 4-Control-Selection
- 5-Control-Implementations
- 6-Risk-Register
- 7-POAM
- 8-Authorization-Summary

## Goal
The goal of this repository is to demonstrate hands-on understanding of:
- RMF concepts
- SaaS shared responsibility
- system scoping and categorization
- security control implementation
- risk management documentation
