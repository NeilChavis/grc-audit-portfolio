# Cloud Responsibility (SaaS Model)
The Cloud File Sharing System (CFSS) operates within a Software as a Service (SaaS) environment, where security responsibilities are shared between the service provider and Apex Consulting Group.

## Service Provider Responsibilities
- Manages underlying infrastructure, including servers, storage, databases, and networking
- Maintains operating systems and platform-level services
- Ensures system availability, backups, and redundancy
- Implements and maintains infrastructure-level security controls

## Apex Consulting Group Responsibilities
- Manages user accounts, roles, and access permissions
- Configures system settings and enforces security features
- Controls how data is stored, shared, and handled within the platform
- Monitors user activity and enforces internal security policies
