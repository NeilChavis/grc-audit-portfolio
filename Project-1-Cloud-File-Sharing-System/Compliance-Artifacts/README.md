# Compliance Artifacts
This folder contains the primary artifacts developed during the security assessment of the Cloud File Sharing System (CFSS).

## Included Deliverables:
### System Security Plan (SSP): Documents system architecture, control implementation, and security posture
### Security Assessment Report (SAR): Summarizes control effectiveness and identified findings
### Risk Register: Captures identified risks, including impact, likelihood, and mitigation strategies
### Plan of Action & Milestones (POA&M): Outlines corrective actions and timelines for addressing control gaps
