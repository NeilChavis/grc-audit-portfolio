# Compliance Artifacts
This folder contains the primary artifacts developed during the security assessment of the Cloud File Sharing System (CFSS).

---

### Included Artifacts

- **System Security Plan (SSP)**  
  Documents the system architecture, security controls, and overall security posture.  
  [View SSP](01%20SSP.pdf)

- **System Boundary Diagram**  
  Visual representation of the CFSS system boundary, including in-scope components and external dependencies.  
  ![System Boundary Diagram](02%20cfss_boundary.png)

- **Risk Register**  
  Identifies and evaluates risks based on impact and likelihood, with mapped controls and mitigation strategies.  
  [View Risk Register](03%20Apex%20Risk%20Register.pdf)

- **Control Selection**  
  Documents selected security controls based on FIPS 200 and NIST SP 800-53B Moderate baseline.  
  [View Control Selection](04%20control_selection.pdf)

- **Security Assessment Report (SAR)**  
  Summarizes control effectiveness and identifies security findings from the assessment.  
  [View SAR](05%20SAR.pdf)

- **Plan of Action & Milestones (POA&M)**  
  Outlines corrective actions, priorities, and timelines for addressing identified control gaps.  
  [View POA&M](06%20POA&M.pdf)
